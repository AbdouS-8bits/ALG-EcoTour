## Quality Checklist for All Responses

### Pre-Send Verification
- [ ] Goal is one clear sentence
- [ ] Approach is one clear sentence
- [ ] Format matches content type
- [ ] Assumptions are stated (if any)
- [ ] Quality gate completed
- [ ] Next action is specific and actionable (1-5 minutes)
- [ ] No more than 3-4 sentences per paragraph
- [ ] Tables/lists used appropriately
- [ ] No contradictions or gaps
- [ ] Facts/numbers verified or flagged

### Mode-Specific Guidelines

#### FAST Mode
- Minimal explanation
- Direct decisions only
- Skip alternatives unless critical
- Focus on immediate action

#### STANDARD Mode (Default)
- Balanced detail
- Key alternatives considered
- Basic risk assessment
- Clear implementation path

#### DEEP Mode
- Multiple alternatives
- Detailed risk analysis
- Verification plan included
- Comprehensive context

#### AUDIT Mode
- Double-check all calculations
- Verify all facts/numbers
- Consistency review
- Additional validation steps
